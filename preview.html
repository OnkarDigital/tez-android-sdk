Behaviour Reference

Powered by TezGatewayHidden when remove_branding = true
Header colorFrom CheckoutSettings.headerColor (hex). Applied to spinner tint + check-status button too.
Google Pay buttonShares QR image to GPay via ACTION_SEND + FileProvider. Falls back to deep link if no QR.
Other UPI buttonsDeep link: phonepe_link / paytm_link / bhim_link / amazonpay_link / cred_link
QR sectionShown when show_qr = true and qr_image is not blank
Status pollingStarts after user returns from UPI app. Polls every 5s for up to 100s (20 attempts)
Early stopStops immediately on SUCCESS or FAILURE — does not wait for 100s
Check Status NowCancels current poll and triggers immediate recheck
Result cardSuccess (green) / Failed (red) / Pending (amber). Auto-dismisses after 1.8s on success, 2s on failure.
News tickerMarquee shown only when settings.news is not blank
