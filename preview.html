Behaviour Reference  ·  v1.0.11

UI Redesign v1.0.11 Gradient amount card, accent section labels, 80dp UPI buttons with brand-tinted backgrounds, ghost cancel button, header drop shadow, trust row.
Amazon Pay / CRED fix FIX show_amazonpay & show_cred were missing from SettingsClient.kt — always defaulted to true. Now parsed from API; default is false.
Trust row NEW Green pill: 🔒 Secure · ⚡ Fast · 📱 Native UPI. No legal claims.
News ticker UPD Yellow 📢 left band with amber text on cream background. Shown only when settings.news is not blank.
Amount card UPD Gradient background (#F7E6FF → #FDFAFF), 38sp bold amount, accent underline. Hidden when paymentData.amount is blank.
Header Lock icon inside translucent circle badge. Drop shadow below header for depth. "✓ NPCI Certified" in subtitle.
QR "I've Paid" button FIX Tap to directly start status polling without leaving the app.
Header color From CheckoutSettings.headerColor. Applied to spinner tint, "Check Status" border, "I've Paid" button.
Status polling Polls every 5s for up to 100s (20 attempts). Stops immediately on SUCCESS or FAILURE.
Result cards Success (green) auto-dismisses 1.8s, Failure (red) 2s, Pending (amber) stays until user closes.
