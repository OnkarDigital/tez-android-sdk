Theme Layout Reference Table  ·  v1.0.22

Theme 1: Classic Classic top-down card layout. Features a purple accent header, custom gradient amount card, 3-column button grid, clean green trust row, and outlined cancel button.
Theme 2: Cyberpunk Dark Premium dark-mode structure. Features a deep charcoal layout body (`#0E0E18`), a sleek horizontal scroll row of UPI apps, glassmorphic cards, neon green trust items, and a translucent red cancel button.
Theme 3: Neo-Brutalism High-contrast, bold design. Features a warm off-white body (`#FAF9F6`), clean white headers/cards with thick black outlines (`2.5dp`), flat brutalist drop-shadow offsets, and a 2-column app button layout.
Theme 4: Royal Gold Luxury VIP theme. Features a pure black background (`#121212`), a vertical list (1 column) of gold-bordered app selectors, deep gold crest header details, gold text, and gold-tinted status bars.
